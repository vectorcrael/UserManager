Welcome to the User Management App

This is a simple Angular website built to help manage users. It uses Bootsrtap for the interface and connects to a file based database. Feel free to play around with it. For the source code, click on the botton below.

GitHub Repo

How to Use the Site

Use the menu above to navagate the site.
For the regular users you are only allowed to view the users and register a new user.
For administrators you can make some changes to the user details so long as you don't break anything!
